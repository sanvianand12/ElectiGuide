ElectiGuide is an interactive roadmap designed to help voters research candidates and understand the election process using Django and JavaScript.
## 🚀 Setup Instructions
To run ElectiGuide locally, follow these steps:
1. Clone the repository: `git clone <your-repo-url>`
2. Create and activate a virtual environment.
3. Install dependencies: 
   ```bash
   pip install -r requirements.txt
4. Run migrations:
   python manage.py migrate
5. Start the server:
   python manage.py runserver
